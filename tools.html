web常用方法汇总

1. 获取地址栏问号及后面的内容

var cont=window.location.search;

2. 获取地址栏问号前面的内容(不包括问号)不包含参数

var cont=window.location.pathname;

3. &&的用法技巧

例子：var cont=null&&1;结果：1;var cont1=23&&1;结果：1

&&连接两边，不管两边是什么值，都返回右边的值

4. ||的用法技巧

例子：var cont=null||1;结果：1;var cont1=23||1;结果：23;var cont1=null||undefined;结果：undefined

||连接两边，如果两边都为假，返回右边的值。如果一真一假，返回真的值。如果两边都为真，返回左边的值。

5.jq操作iframe（服务器环境下测试）

例子：子页面操作父页面元素

$(window.parent.document).find('body')....

例子：父页面操作子页面元素

$(window.frames["iframe1"].document).find("")...

6.低版本ie浏览器清除浮动(不添加标签)

给父页面加上  overflow: hidden;_zoom:1;

7.操作数组的方法之splice

splice方法向数组中删除元素(2个参数时，有返回值，就是所删除的元素)或添加元素（3个参数），并且改变了原数组

例子splice：arr_1=arr.splice(2,2); 从2开始删除2个元素，改变了原数组arr,并把截到的值返回给新数组arr_1

例子splice：arr.splice(2,1,"William"); 从2开始删除1个元素 ，再在此位置增加一个元素'william'

8.操作数组的方法之slice

slice方法从已有的数组中返回选定的元素,不改变原数组

例子slice：arr_2=arr2.slice(2,5);//从2开始截到5(不含),并把截到的值返回给arr_2

9.获取两个数之间的随机数

function ran(n,m){ return Math.floor(Math.random()*(m-n)+n); }
